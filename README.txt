# Hercules Obfuscator - Private Bot & Web Platform

تم تحديث وتجهيز هذا المجلد ليدعم تشفير الأكواد في الخاص (Direct Messages) لحماية خصوصية المطورين والأكواد، مع بقاء منصة الويب والـ API جاهزة للرفع على Railway.

## 📂 محتويات المجلد:
1. `api/obfuscate.js`: الكود المحدث لحصر الأوامر في الخاص وحماية السيرفرات العامة.
2. `package.json`: الاعتماديات المطلوبة للتشغيل.
3. `.env`: ملف إعدادات التوكن المحلي.

## ⚙️ طريقة التثبيت والتشغيل المحلي:
1. فك ضغط هذا المجلد وانقل محتوياته إلى مجلد مشروع التشفير الرئيسي الخاص بك.
2. افتح منفذ الأوامر ونفذ: `npm install`
3. شغل المشروع عبر: `npm start`
